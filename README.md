# 🚀 Job Progress Tracker

A web application that solves the challenge of managing workers, tasks, attendance, and payments in one centralized system. It helps employers track job progress efficiently while giving workers clear visibility into their tasks, work records, and earnings.

---

## 📌 Overview

Managing workforce operations manually or across multiple tools leads to inefficiency, poor tracking, and lack of transparency.

This system provides:
- A single platform for managing jobs and workers
- Clear tracking of daily work progress
- Organized attendance and payment records

---

## ✨ Features

- 👥 Worker Management  
- 📁 Project & Task Tracking  
- 🕒 Attendance Monitoring  
- 💰 Payment Tracking  
- 📊 Reports & Insights  
- 🔔 Notification System  
- 🔐 Role-Based Access (Employer / Worker)
